# 台股 AI 買賣決策系統

純前端（HTML5 / CSS3 / ES6）＋ LocalStorage 的台股**決策輔助**系統（非下單系統）。
可直接部署 GitHub Pages，手機與電腦皆可使用。

## 功能模組
- 🏠 儀表板：市場溫度 / 風險指數 / AI 市場評分儀表 / 損益總覽 / 資產曲線
- 🔍 選股中心：自建股池（均線、RSI/KD/MACD、三大法人籌碼、融資）
- 🧱 五鐵律評分：5 條件各 +20，綠/黃綠/黃/橘/紅燈
- 🩸 進場引擎：低點抄底指數 + 進場建議 + 風險等級
- 🔴 出場引擎：高點逃命指數 + 減碼 / 停利建議
- 🎯 AI 目標價：黃金分割 + 平台測幅，三段目標價
- 🛡 停損資金：停損價 / 最大虧損 / 建議張數 / 風險報酬比
- 💼 持股管理：即時損益，一鍵平倉轉交易紀錄
- 📒📊 交易紀錄與統計：勝率 / 淨利 / 連勝連敗 / 獲利因子
- 🤖 AI 交易教練：紀律自評 + 每日評分 + 改善建議
- 🐯 AI 股票人格：猛虎 / 獵豹 / 盾牌 / 火山 / 成長 自動分類

## 使用方式
直接用瀏覽器開 `index.html`；或部署到 GitHub Pages 後開啟網址即可。
資料儲存在瀏覽器 LocalStorage，可用右上角按鈕匯出 / 匯入 JSON 備份。

## 技術
HTML5 / CSS3 / JavaScript ES6 / Chart.js（CDN）/ LocalStorage，無需後端。

> ⚠️ 本系統僅為技術分析與交易紀律的輔助工具，所有數據需自行輸入，不構成投資建議。
